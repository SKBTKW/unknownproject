# 03-03. 開発ルール（Development / ニコイチ）

## 概要
同じベース地形の土地を2枚重ねて強化するシステム。圧縮と成長を両立させるための重要な仕組み。

## 基本ルール
- **対象**: 同じベース地形同士のみ（平地＋平地、丘陵＋丘陵など）
- **コスト**: 🔥残り火を **2** 消費（通常の土地配置は1）
  - 属性付き土地を開発する場合はさらにコストが増加する可能性あり（検討中）
- **効果**:
  - ベース地形の出力が強化される
  - 属性を最大 **2つ** まで引き継げる
  - 3つ以上の属性がある場合は、任意で2つを選択（残りは破棄）

## 属性の扱い
- 同一属性が重複した場合 → ボーナスが付与され、1つに統合可能（例: 森の効果強化）
- 異なる属性の場合 → 最大2つまで選択して保持

## 世界観的な解釈
同じような土地を開拓・統合することで、より豊かで防衛力の高い土地に成長させる。
