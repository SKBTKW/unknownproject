# 10. 政体（Polity）

## 概要

政体は**ゲーム全体に影響を及ぼす国家の基本方針**を表すシステム。
社会制度ツリーを通じて複数の政体を取得し、状況に応じて1つをアクティブにすることで、プレイスタイルを柔軟にカスタマイズできる。

## 基本的な仕組み

| 項目 | 内容 | 備考 |
| ------ | ------ | ------ |
| **開始時** | 弱いデフォルト政体を保持 | 「ベース土地を発見しやすくなる」などの軽い効果 |
| **取得方法** | 社会制度ツリーで複数の政体を獲得可能 | 取得自体は比較的自由 |
| **アクティブ数** | **常に1つだけ** | 同時に複数の政体をアクティブにすることはできない |
| **アクティブ化コスト** | 🔥を消費 | 初めてアクティブにする時に必要 |
| **切り替えコスト** | 🔥を消費（やや高め） | すでに別の政体がアクティブな状態で変更する場合 |
| **維持コスト** | なし | アクティブにしている間は追加で🔥を消費しない |
| **切り替えのしやすさ** | 任意のタイミングで可能 | ただしコストが高いため、頻繁な切り替えは抑制される |

## 設計の意図

- 政体は「今どの方向性を強化したいか」をプレイヤーが状況に応じて選ぶためのシステムとする。
- 社会制度ツリーを進めることで選択肢が増え、徐々に自分の国家の方向性を固めていく。
- 切り替えコストをやや高めにすることで、「気軽に変えすぎない」という緊張感を残す。
- 維持コストをなくすことで、プレイヤーの管理負担を抑えつつ、戦略的な判断の余地を残す。
- 場の構成や自分のプレイスタイルに応じて、柔軟に方向性を切り替えられるようにする。

## 保留・未確定事項

- 具体的な政体の種類と効果内容
- 各政体の強さのバランス
- 社会制度ツリー内での政体カテゴリの位置づけと取得コスト
- デフォルト政体の具体的な効果
- 政体とロールの相性（将来的に考慮する可能性あり）
