# 03. 第2スロット ボーナス資源特性

## 現在のボーナス資源一覧（暫定）

| ボーナス資源 | 主な効果 | 付与条件（主なもの） | レアリティ目安 |
|--------------|----------|----------------------|----------------|
| **果実**     | 🌾 +10〜15 | 平地、森、密林       | コモン         |
| **穀物**     | 🌾 +15     | 平地                 | コモン         |
| **家畜**     | 🌾 +15     | 平地                 | コモン         |
| **鉱脈**     | 🧱 +15     | 丘陵、山岳           | アンコモン     |
| **貴金属**   | 🧱 +10 / ✨ +5 | 丘陵、山岳         | レア           |
| **高級木材** | 🧱 +20     | 森                   | アンコモン     |
| **水産物**   | 🌾 +15     | 川、湖               | アンコモン     |

## ルール
- 1土地につき**最大1つ**まで付与可能（第2スロット）
- 属性（第1スロット）と組み合わせて強い土地を作るのが基本戦略
- イベント「土地特性付与」カテゴリで主に獲得する

## 設計意図
第2スロットを設けることで、土地の個性をさらに細かくし、「この土地をどう育てるか」の選択肢を増やしている。
特に貴金属のようなレアなボーナスは、長期的なアドバンテージになる。
