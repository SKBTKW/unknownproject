# 03. 土地システム（開発・保留含む）

このセクションでは、土地の種類、特性の付与ルール、開発（ニコイチ）、保留ゾーンの仕様をまとめています。

## ファイル構成
- `01_base_terrains.md` : ベース地形（平地・丘陵・山岳・湖）の基本出力
- `02_attributes.md` : 第1スロット属性（森・密林・砂漠・川）の仕様
- `03_development.md` : 開発（ニコイチ）ルール
- `04_reserve.md` : 保留ゾーンルール
- `05_bonus_resources.md` : 第2スロットボーナス資源特性

## 現在の設計方針
- 土地は「ベース地形 + 第1スロット属性 + 第2スロットボーナス資源」の3層構造
- 開発は**同じベース地形同士**でのみ可能
- 保留ゾーンは最大3スロット。ターン開始時に使用中であれば🔥を1消費
- 場の土地構成がドローの傾向に影響を与える（特に土地発見・特性付与カテゴリ）
