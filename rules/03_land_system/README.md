# 03. 土地システム

## 概要
土地は「ベース地形 + 属性 + ボーナス資源」の3層構造で構成される。

## ファイル構成
- `01_base_terrains.md` : ベース地形の基本規則
- `02_attributes.md` : 第1スロット属性の仕様
- `03_bonus_resources.md` : 第2スロットボーナス資源の仕様
- `04_legendary_terrains.md` : レジェンダリー地形
- `05_development.md` : 開発ルール
- `06_reserve.md` : 保留ゾーンルール
- `007_palace.md` : 宮殿について

## 重要ルール
- そのターン中に新しく配置した土地は、そのターンは収入が発生しない
- 開発は同じベース地形同士で可能
- 保留ゾーンは最大3スロット
- **保留中の土地は場に出るまで効果が発生しない**
- 場の土地構成がドローの趨向に影響を与える
