# レジェンダリ地形

このファイルでは、通常の土地とは異なる特別な地形（レジェンダリ地形）について記載する。

## 海 (Sea)

**位置づけ**  
人類が世界の片隅で生き延びる中で、初めて到達した「未知の果て」。  
長い探索と投資の末に手に入れる、圧倒的なロマンと力を秘めた特別な存在。

**取得タイミング**  
- 最速：35ターン目以降  
- 現実的な取得タイミング：37〜42ターン目  
- 川探索イベント発生後、最低12〜15ターンの過程が必要

**性能の方向性**  
- かなり強力な産出と特殊効果を持つ  
- ただし場のスロットを2つ消費するなど、重いコストを伴う  
- 恩恵を受ける期間が長くない（取得が遅いため）

**設計思想**  
- 人類が初めて到達する「未知の果て」としてのロマン  
- 長い焦らしの末に手に入れる達成感  
- 次のプレイでも「また海が欲しい」と思わせる欲求の残し方
