# 01. 全体コンセプト

## ゲーム概要
- **タイトル**: マジェスタ (Majestic Stand: Last Ember)
- **プレイ時間**: 約50ターン固定
- **プレイヤー数**: 1人（シングルプレイヤー）
- **ジャンル**: ローグライク要素を持つ箱庭内政ストラテジー + カード構築的要素

## 世界観
人類は強大な亜人種たちに世界の大部分を奪われ、わずかな土地に追いやられている。
人々は「最後の残り火（🔥）」を象徴とする活力を頼りに、3度にわたる破滅的な試練を耐え抜き、人類の存続の道を切り開く。

## 勝利条件 / 敗北条件
- **敗北条件**: 🔥（残り火）が0になる
- **勝利条件**: 50ターン終了時に🔥が残っている + 3度の試練を経験する（成否は問わない）
- 試練に失敗しても即死せず、ゲームは継続する（ただし大きなペナルティが発生する可能性あり）

## 設計の核となる思想
「自分の選択が気持ちよく反映される」ことを最優先に、
- 理不尽な事故（特に土地事故）を極力排除
- 試練で負けても即死しない
- 50ターンスキップは0点評価とする
- ネタプレイ・変態プレイも許容する
- 「好き勝手に遊べる余白」を残しつつ、一手の重みを感じられる緊張感を出す

## ゲームの流れ（大まか）
1. ロール選択（将軍 / 預言者 / 開拓者）
2. 毎ターン：ドロー → 判断 → 行動（🔥を消費）
3. 約15ターン前後、30ターン前後、50ターンで試練発生
4. 試練を経験しつつ、50ターンを終える

## 現在の開発優先度
1. 土地システムの完成（ベース地形・属性・開発・保留）
2. ドローシステムの完成
3. 試練システムの詳細化
4. ロールごとの差別化
5. 社会制度ツリー・建築物
