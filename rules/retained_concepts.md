# 保留コンセプト一覧

このファイルは、現在保留中または未確定の事項をまとめたものです。

## 1. 川・海関連（最優先保留）

### 川イベント
- 発生確率と発生条件の最終調整
- 各回の援助コストの具体的な数値
- 成功率の詳細設計（援助量との関係）
- 探索隊の便りの具体的な表現と情報開示レベル
- 1プレイあたりの発生回数制限の扱い

### 海（レジェンダリー地形）
- 具体的な産出値（食料・資材・防御力）
- 特殊効果の内容
- 場のコストの詳細（スロット2つ消費以外の要素）
- 取得時の演出・テキスト
- 到達タイミングの微調整（35〜40ターン目）

## 2. 視角・表現関連

- ネガポジ属性によるカードフレーム変化の詳細仕様
  - ネガティブ：寒色系の祸っぽい感じ
- レアリティによる視角的変化の詳細
  - レジェンダリー：神々しい表現
- カードのツールチップの活用方法

## 3. 災害・危機関連

- 各災害の具体的な数値（下降ターン数、失う量など）
- 前兆イベントの詳細設計
- 災害発生時の演出・テキストのトーン
- 「敢えて災害を選ぶ」選択肢の有無とバランス

## 4. その他の保留事項

- 社会制度ツリーの詳細構造
- ロール固有のスキルツリーの有無と内容
- 暗黒時代（苦難の時代）の仕様
- 初回ボーナス・市民クエストなどの序盤演出
- 土地開発時の特性引き継ぎルールの最終調整
- 保留ゾーンの拡張方法
- 建築物の必要性・解禁条件・効果
- 神秘リソースの具体的な使い道とバランス
- 試練時の戦術カードの詳細設計

## 5. 将来的に検討する要素

- 交易システム
- 軍事・兵士システムの詳細
- エンディングの分岐条件
- プレイスタイル別のバランス調整
