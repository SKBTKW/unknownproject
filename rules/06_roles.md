# 06. ロール

## 3つのロールと特徴

### 将軍（General）

**通常時パッシブ** :

+ 軍備カテゴリの出現率がやや上昇
+ 防衛関連施設のコストが軽減

**試練時能力** :

+ 試練期間中、防衛力にボーナス
+ 強力な戦術カードを優先的に使用可能
+ 「少数精鋭で耐える」プレイに適したロール

### 預言者（Prophet）

**通常時パッシブ** :

+ ✨神秘の産出量 +25〜35%
+ 神秘蓄積の要求量がやや軽減（ツリーでさらに強化可能）

**試練時能力** :

+ 試練開始の7ターン前から内容をある程度把握可能
+ 試練期間中、神秘の効果が大幅に強化
+ 情報優位と神秘を活かした「トリッキーな対応」が強み

### 開拓者（Pioneer）

**通常時パッシブ** :

+ 土地発見カテゴリの出現率が上昇
+ 開発コストがやや軽減

**試練時能力** :

+ 試練期間中、🔥消費が軽減（1/2など）
+ 場を素早く強化しやすい
+ 「土地をたくさん育てて耐える」堅実なプレイに強い

## 設計意図

各ロールで「普通に遊んでもそれなりに強いが、特徴を活かせばより快適になる」バランスを目指す。
特に預言者は神秘を「ロマン枠」ではなく「明確な戦略的選択肢の一つ」として成立させることを目標としている。
