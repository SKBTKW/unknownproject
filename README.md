# マジェスタ (Majestic Stand: Last Ember)

人類が世界の片隅に追いやられた世界で、**最後の残り火（🔥）**を絶やさないよう、3度の試練を乗り越える50ターン固定のストラテジーゲーム。

## 設計思想（最優先原則）
- MTGのような土地事故は絶対に避ける
- リプレイ性が高い
- 理不尽なゲームオーバーは存在しない（🔥が0になった時のみゲームオーバー）
- 自分の選択が気持ちよく反映される
- ルールは理解しやすい
- バフやトークンを快適にコントロールできる
- 楽しい取捨選択のジレンマがある
- 「好き勝手に遊べる余白」を残しつつ、一手の重みを感じられる緊張感を出す

## 現在のフォルダ構成
- `rules/` : ルール詳細（セクションごとに分割）
- `parameters/` : 数値パラメータ（YAML形式で管理予定）
- `CHANGELOG.md` : 変更履歴

## 進行状況
- 土地システム（ベース地形・属性・開発・保留）を優先的に固めている
- ドローシステムも最新仕様に更新済み
- 試練・ロールについては現在整理中
